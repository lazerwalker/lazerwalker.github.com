Flappy Bot / Tinsel

Call (646) 681-7902 to Play.

Tinsel is a game engine I created to make it easy to create interactive fiction that can be played from any touch tone phone. Writing simple Ruby scripts, you can make games that use the dial pad for input and text-to-speech as output.

The phone number listed above is a clone of the game Flappy Bird. Call it and give it a try!
