ClojureScript Koans

Role: UX Architecture, Visual Design, Code

Code koans, a set of small exercises designed to teach you concepts and ideas in isolated, atomic units, are a popular way for experienced programmers to learn new languages.

Most code koans are very poorly-designed, though. They frequently require you to set up a full development environment on your local computer, which can be time-consuming. They then typically present you with a single text file containing dozens of exercises, which can be daunting for newcomers. A few koans have online versions, but they mirror this frustrating format.

I set out to make a set of code koans that provided a more game-like experience to make learning a new language more fun.

Specifically, they feature:
  • An over-the-top minimalist zen aesthetic that's more fun and inviting than a plain text editor.

  • A focus on a single koan at a time, making them feel more like completing levels in a game than exercises in a worksheet.

  • Instant real-time feedback, by means of teaching a JavaScript-based language that can thus be compiled and run completely in the browser.

Since their release, the ClojureScript Koans have become one of the most popular ways to learn ClojureScript.

They are available online at http://clojurescriptkoans.com.
